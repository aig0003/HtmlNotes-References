CSS Practice

To start with CSS, you first need to create a .css file, and link it to a desired HTML file, this can be done via a link tag (<link>) in the head (<head>) section of the HTML. The link tag should have 3 attributes:

  • rel="stylesheet"
  • type="text/css"
  • href="LinkToYourCSSFile.css"

Colors in CSS

There are many ways of introducing and assigning color into your HTML/CSS. You can change the color of an element in CSS using the "color" attribute.

  1. There are many named, predetermined colors to use
    • This can be something as simple as red, green, blue, etc.
    • Or less common colors, like CornFlowerBlue, DarkOrchid, MediumTurquoise, etc.
    • You can find a list of all 140 named CSS colors here.
  2. Another way is to use base 16, hexadecimal values to determine color, which allow near unlimited customizable colors Note that hex values are preceded by an octothorp (#) symbol.
    • This is #A97B20
    • This is #FF1493
    • It's good to remember that in hex colors the digits correspond to a RGB value, meaning the first two digits correspond to a red value, the next two to a green value, and the last two to a Blue value (# R R G G B B).
  3. You can also use RGB channels, which is made of 3 channels: Red, Green, and Blue, with each value of those channels being 0-255.
    • This is rgb(0, 255, 0).
    • This is rgb(50, 100, 150).
    • This isn't necessarily better or worse than using hexadecimal format, its mainly a different syntax which may or may not be easier to understand.
  4. Finally, there is RGBA, which is similar to the RGB method, except it has an additional channel, alpha, which determines transparency from decimal values ranging from 0 to 1
    • This has full transparency rgba(0, 0, 255, 1)
    • This has half transparency rgba(0, 0, 255, .5)
    • This has 1/10 transparency rgba(0, 0, 255, .1)
