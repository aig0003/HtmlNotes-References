HTML Tags

  • Images (<img>) are self-closing tags which use a source (src) link to display either a local or hosted image
  • Links, or anchor tags (<a>) are inline elements, not self-closing, and require a href to link to a page. Note that if you want to link to an internet address, you need to include the http:// link, otherwise the browser will think you are linking to local files.

Tables

Table elements (<table>) consist of table rows (<tr>), and those rows are populated with table data (<td>).

To create headings, you would populate the first table row with table headers (<th>), instead of table data

A slightly more robust way of making tables in HTML5 is to create a table head (<thead>), and a table body (<tbody>) and populate them with the appropriate table rows.

<tbody> data 1 <tbody> data 2

Forms, Inputs and Validations

Here we're going to focus on 3 main elements, the form (<form>), inputs (<input>), labels (<label>), and simple validation. (Note that the following examples are not going to be function, simce this is meant to be a pure HTML reference, and a back end will not be present here).

Forms and Basic Inputs

Form tags have two major attributes: action, which specifies the address the form data is sent to, and method, which is the type of HTTP request.

Inputs types are determined by specifying the Type attribute. Name attributes can also be specified in the input tag, which will help for query requests on submission.

Inputs can be nested inside of label tags (<label>), which are useful for impaired browsing and appropriate assignments.

Basic Validations

Some basic validations to form input can be done with basic HTML. Some common examples are ensuring a non-empty text empty (using the Required attribute), Emails can be validated by setting type to email, and passwords can have a character limit via a pattern attribute with a title attribute outlining the requirements.

Radio Buttons, Dropdowns, and Text Areas

By giving Radio-type inputs the same Name attribute, you can limit a single choice to a group of radio buttons by with that name.

For inputs to be useful, you need to set a Value attribute for the form to return the results with.

To create a dropdown menu, you want to use a select tag (<select>), populated with option tags (<option>). To a return a value, set a Name attribute in the select tag, and set the desired return values via the Value attribute in the option tags

Text areas can be created via the text area tag (<textarea>), and set to a specific size by setting the Rows and Cols attibutes.
