Basic text and Paragraphs

I can either write text without any tags,

or, I can us the <p> tag to denote a block-element pargraph.

Note: notice the in-code use of strings "&lt;" and "&gt;" to act as escape characters to display the less than or greater than signs (and in this sentence, "&amp;" being used for the ampersand), without confusing the browser into thinking they are tags.

You can create inline bold, italicized, underlined, or strikethrough text using <b>, <i>, <u>, and <s> tags, respectively (although this isn't necessarily the best way to do this anymore).

Since HTML5, It's better to bolden and italicize characters using the <strong> and <em> (emphasis) tags, since these denote the actual structure of the HTML and not their styling.

HTML Tags

  • Images (<img>) are self-closing tags which use a source (src) link to display either a local or hosted image
  • Links, or anchor tags (<a>) are inline elements, not self-closing, and require a href to link to a page. Note that if you want to link to an internet address, you need to include the http:// link, otherwise the browser will think you are linking to local files.
